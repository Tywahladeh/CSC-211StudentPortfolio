Basic Tag Usage

Tags

HTML tags are elements used to define the structure and content of a web page. They are composed of an opening tag, content, and a closing tag.
Tags are used to describe various parts of a document, such as headings, paragraphs, links, images, lists, and more. HTML tags provide the structure and semantics for web content, allowing browsers to render it appropriately. They are fundamental to web development and are used in conjunction with other technologies like CSS (Cascading Style Sheets) and JavaScript to create interactive and visually appealing web pages.

List of tags

  • link: Links external resources like stylesheets (CSS) or icons.
  • script: Embeds or references JavaScript code.
  • body: Contains the visible content of the document.
  • header: Represents introductory content at the beginning of a section
  • footer: Represents a footer for a section or the document.
  • nav: Defines a set of navigation links.
  • section: Groups related content together.
  • article: Represents an independent piece of content in a document.
  • aside: Contains content that is related to the main content but not essential.
  • main: Contains the main content of the document.
  • a: Creates a hyperlink to another page or resource.
  • img: Embeds an image.
  • ul, ol: Create unordered (bulleted) and ordered (numbered) lists.
  • li: Defines a list item in a list.
  • div: A generic container for grouping elements and applying styles.
  • span: A generic inline container for applying styles.
  • table: Defines a table.
  • tr: Defines a table row.
  • th: Defines a table header cell.
  • td: Defines a table data cell.
  • form: Represents an interactive form for user input.
  • input: Creates an input field for forms.
  • button: Creates a clickable button.
  • label: Labels form elements.
  • textarea: Defines a multiline text input field.
  • select: Creates a dropdown list.
  • option: Defines an option in a select element.
  • iframe: Embeds an external webpage within the current document.
